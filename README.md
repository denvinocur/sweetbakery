# Sweet Bakery 🍰

Proyecto final desarrollado para el curso de Front-End.

## Descripción

Sweet Bakery es un sitio web de una pastelería online donde los usuarios pueden explorar un catálogo de productos, buscarlos por nombre, filtrarlos por categoría y agregarlos a un carrito de compras.

Además, cuenta con un formulario de contacto conectado mediante Formspree para el envío de consultas.

## Funcionalidades

- Catálogo dinámico de productos.
- Búsqueda por nombre.
- Filtro por categorías.
- Carrito de compras con persistencia mediante LocalStorage.
- Validación de formulario con JavaScript.
- Envío del formulario mediante Formspree.
- Diseño responsive utilizando Bootstrap 5.

## Tecnologías utilizadas

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Fetch API
- JSON
- LocalStorage
- Formspree
