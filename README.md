# Preentrega Front End

Este proyecto consiste en una página web de una pastelería desarrollada con HTML y CSS.

## Características

- Estructura semántica HTML.
- Diseño responsivo.
- Uso de Flexbox y Grid.
- Formulario funcional con Formspree.
- Navegación interna.
- Integración multimedia.
- Estilos externos con CSS.

## Tecnologías utilizadas

- HTML5
- CSS3
- Google Fonts
- Formspree
